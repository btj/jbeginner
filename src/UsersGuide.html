JBeginner User's Guide

JBeginner is a tool for teaching and learning the Java programming language at the introductory level. It enables the user to construct simple programs by selecting program elements from menus, and to step through them. All constructed programs are well-typed and there are no compiler errors; in fact, there is no compilation. While a user is stepping through the program, the tool displays the objects on the heap, the frames on the stack, and the variables they contain. The tool is itself written in the Java programming language.

How to construct and run a program

  • Add classes using the Add Class... command in the Program menu.
  • Right-click on the header of a class to add fields, methods, and constructors.
  • Right-click on the header of a constructor, method, or compound statement to add or remove statements.
  • Right-click on an expression to change it.
  • Right-click on the header of a constructor of a class to create an object of the class.
  • Right-click on the header of a static method to invoke the method.
  • Right-click on an object to invoke an instance method.
